Welcome to the SCRAPING CLUB!

How to use:

Instagram

  • scrape instagram profile by entering this address
    {web-domain}/instagram/profile/{ig_username}
  • scrape instagram profile post by entering this address
    {web-domain}/instagram/post/{ig_username}
  • scrape instagram hashtag post by entering this address
    {web-domain}/instagram/post/hashtag/{hashtag_name}

Facebook

  • scrape facebook profile by entering this address
    {web-domain}/facebook/profile/{fb_username}
  • scrape facebook page post (public page) by entering this address
    {web-domain}/facebook/page/{page_name}
  • scrape facebook group post (public group) by entering this address
    {web-domain}/facebook/group/{group_name}
